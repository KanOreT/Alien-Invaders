# Текстовий емулятор "Нашестя інопланетян" 👾

Це покрокова консольна гра на Python, створена в рамках навчання.

## Як запустити:
1. Встановіть Python.
2. Завантажте файл `main.py`.
3. Запустіть через термінал: `python main.py`.

## Керування:
- `a` — вліво
- `d` — вправо
- `w` — постріл
- `Enter` — пропустити хід
